## 「為何被拒絕『加功能』、『改功能』？」

這隻程式的寫法已經與現今 Android InputMethodService 運作方式出入甚大了，現在做 Android 輸入法，連按鍵都要自己刻，沒有 `<Key>` 這種現成元件，其餘改動更是多如牛毛。Android 內部若已完全禁用或不再相容舊版寫法，我也愛莫能助。
